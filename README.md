# LootGen

LootGen — консольная утилита на Python для симуляции выпадения редких игровых предметов (лута) из сундуков и оценки эмпирических вероятностей дропа.

## Описание

Приложение моделирует открытие заданного количества сундуков и считает, сколько раз выпал каждый предмет из пула. Результаты выводятся в виде форматированной таблицы в терминале с помощью библиотеки `rich`. Это позволяет быстро оценивать баланс вероятностей и использовать утилиту при разработке и настройке внутриигрового лута.

## Технологии

- Python 3.11+
- `rich` — красивый вывод текста и таблиц в терминал
- `flake8` — статический анализ и контроль качества кода
- Git + GitHub
- GitHub Actions — автоматический запуск линтера при `push` и Pull Request

## Структура проекта

- `lootgen.py` — основной скрипт утилиты (логика симуляции и вывод таблицы).
- `requirements.txt` — зафиксированные версии зависимостей (`rich`, `flake8`).
- `.gitignore` — исключение служебных файлов, виртуальных окружений и артефактов сборки.
- `.flake8` — конфигурация линтера (ограничение длины строки, исключения директорий и т.д.).
- `.github/workflows/ci.yml` — конфиг GitHub Actions для автоматического запуска `flake8`.
